# Previous Releases

This folder contains older versions of the OpenGrok MCP Extension.

⚠️ **These are archived versions** - For the latest version, use the `.vsix` file in the parent directory.

## Version History

- **1.0.0** - Initial release
- **1.0.2** - Early updates
- **1.0.3** - Bug fixes

## Installing Old Versions

If you need to install a specific older version:
1. Right-click on the `.vsix` file
2. Select "Install Extension VSIX" in VS Code
3. Or use: `code --install-extension <vsix-file>`

**Note:** Always prefer the latest version unless you have a specific compatibility requirement.
